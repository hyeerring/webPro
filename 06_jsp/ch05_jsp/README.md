## [05] JSP

|            |                                  |                                  |
| ---------- | -------------------------------- | -------------------------------- |
| 지시자     | <%@    %>                        | 페이지 속성                      |
| 주석       | <%--    --%>                     | 응답 페이지에서 보이지 않는 주석 |
| 선언       | <%!    %>                        | 전역변수 및 메소드 선언          |
| 표현식     | <%=    %>                        | 결과값 출력                      |
| 스크립트릿 | <%    %>                         | Java 코드                        |
| 액션태그   | \<jsp:action\>    \</jsp:action> | 표준액션태그                     |
|            | \<c:set>    \</c:set>            | 커스텀액션태그                   |
